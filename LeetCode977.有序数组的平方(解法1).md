977.有序数组的平方
给定一个按非递减顺序排序的整数数组 A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。
示例 1：
输入：[-4,-1,0,3,10]
输出：[0,1,9,16,100]

示例 2：
输入：[-7,-3,2,3,11]
输出：[4,9,9,49,121]

提示：
1 <= A.length <= 10000
-10000 <= A[i] <= 10000
A 已按非递减顺序排序。

//解法1：
//思路：排序--创建一个新的数组，它每个元素是给定数组对应位置元素的平方，然后排序这个数组。
class Solution {
    public int[] sortedSquares(int[] A) {
       int N = A.length;
        int[] ans = new int[N];
        for (int i = 0; i < N; ++i)
            ans[i] = A[i] * A[i];

        Arrays.sort(ans);
        return ans;
    }
}
//时间复杂度：O(NlogN)，其中 N 是数组 A 的长度。
//空间复杂度：O(N)。
